# Haunted House THREE JS

## React + TypeScript + Vite

## Установка

1. Устанавливаем зависимости (npm):

   ```
   npm install
   ```

2. Запуск:

   ```
   npm run dev   # Запуск в режиме разработки
   npm run build # Сборка проекта
   npm run preview # Просмотр собранного приложения
   ```

## Стек технологий

- Vite
- Three js
- Javascript
